Corral

Overview

        Corral is a top-down 2D puzzle/strategy game where players need to complete a very specific set of actions in order to group animals into specific positions. The islands you are standing on can be rotated and repositioned, however it is up to you to figure out the best orientation. This project is currently in progress and more details will be coming soon.

Tech Stack
