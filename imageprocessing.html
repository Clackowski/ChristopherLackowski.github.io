Image Processing Program

Overview

        At Northeastern University I took a course named Object Oriented Design where we learned about the general principles of object oriented programming, specifically in Java. In this course we were tasked with creating a program that allows the user to input images of many different filetypes, apply filters to the images and then save them back onto the user's hard drive.

        Additionally, I created a UI to make it easier for the user to use the program. The program supports many different image filters such as colorgrading, mosaic filtering, blurring, mosaic kernels, etc. Sufficient unit testing was performed in order to ensure proper functionality.
