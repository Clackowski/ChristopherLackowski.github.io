Rube Goldberg Machine

Product Demo

Overview

        During my first semester on campus at Northeastern University, I took a class called Cornerstone of Engineering where we learned about many general engineering principles and proper design processes. Half way into this course we were sent home due to the global pandemic in 2020, meaning that we could not complete a project we had been working on in the first half of the course. Therefore, we were given a task while at home. This task was to create a Rube Goldberg Machine that performs some standard daily task. Rube Goldberg Machines are contraptions that carry out a long sequence of small events leading up to an eventual larger action. The contraption that I created using many objects found around a standard household to squirt hand sanitizer into the user hand considering this was relavent to what was going on in the year.

        In my Rube Goldberg Machine, I utilized multiple different hardware and software components. I used an Arduino Uno board along with a breadboard for custom wiring to hardware peices. I used a ultrasonic distance sensor, photoresistor and 2 servo motors. These were programmed using C++ code written in the Arduino IDE. An in-depth technical report was written about the design process and end result which can be found linked below.
