SymbolSense AI

Overview

        At Northeastern University I took a class named Introduction to Artificial Intelligence and I learned a lot about various type of AI such as reinforcement learning, machine learning and complex search algorithms. In this class, I was tasked with completing a final project of my choice, so I decided to develop and train and model that is able to take handwritten documents as input and output the documents as computer text files. This would be completed through machine learning and the model will be trained to be able to separate symbols on a page and then classify them based on given input data.

        For this project, I utilized TensorFlow which is an open source software library for machine learning. It provides many different useful functions and also training data which were extremely important for this application. I used over 250,000 training images which included an image of a handwritten symbol (letter, digit or mathematical symbol) and the name of the symbol that is in the image. Using all of this data, the model was able to be trained and then used to predict when given input images. The final product was extremely accurate, however there were cases where the model predicted symbols incorrectly. In order to improve the accuracy, more data can be given to the model to train with, but this is an expensive task. The technical report and GitHub repo for this project can be found below.
